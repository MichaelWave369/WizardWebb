# Security Policy

## Supported versions
This project is small and community-maintained. Security fixes are applied to the latest `main` branch.

## Reporting a vulnerability
If you believe you found a security issue:
- Do **not** post exploit details publicly.
- Open a GitHub issue with minimal details, or contact the maintainer privately if contact info is provided in the repo.

## Notes
WizardWebb is a link directory. Avoid adding server-side scraping/crawling features that could be misused.
