# Contributing to WizardWebb

Thanks for helping improve WizardWebb!

## Quick start
1. Fork the repo
2. Create a feature branch
3. Run the app locally and make your changes
4. Open a PR

## What to contribute
- New tools/links (edit `data/links.yml`)
- UI improvements (noir vibe, accessibility, mobile polish)
- Categories/tags cleanup
- Bug fixes

## Development
```bash
python -m venv .venv
# Windows:
.venv\Scripts\activate
pip install -r requirements.txt
python app.py
```
